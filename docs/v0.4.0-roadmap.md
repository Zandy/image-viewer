# Image-Viewer v0.4.0 需求规划

## 1. 功能增强

### 1.1 1:1 显示按钮
- 工具栏添加 "1:1" 按钮
- 点击直接切换到原始尺寸显示
- 快捷键：`1` 或 `Ctrl+1`

### 1.2 缩放控件增强
- 工具栏添加 `+` / `-` 按钮
- 或下拉菜单选择百分比（10%, 25%, 50%, 75%, 100%, 150%, 200%, 400%, 800%, 1600%）
- 或可输入的缩放百分比框（支持直接输入数值）
- 当前缩放比例实时显示（如："100%"）

## 2. 多语言支持（i18n）

### 2.1 架构设计
- 使用 Rust i18n 库（如 `fluent` 或 `i18n-embed`）
- 语言文件格式：`.ftl` (Fluent) 或 `.json`
- 目录结构：`locales/en-US/`, `locales/zh-CN/`

### 2.2 第一阶段：中英文双语
- 所有UI文本提取到语言文件
- 支持语言切换（菜单或配置）
- 默认跟随系统语言，可手动设置

### 2.3 扩展性保留
- 语言文件标准化，便于添加新语言
- 支持运行时语言切换（无需重启）
- 预留其他语言接口
